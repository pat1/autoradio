L'interfaccia grafica di interazione con il player dispone una serie di bottoni e icone che possono mostrare degli sti del player e/o impartire comandi.
  • Prima riga:
    • bottone con cartellina: permette l'aggiunta di file o playlist nella posizione del cursore. E' possibile selezionare un solo file; pper inserire più file alla volta utilizzare una playlist in formato m3u o xspf.
    • secondo bottone: mostra lo stato del player; Playing in verde; Paused in azzurro; Stopped in rosso.
  • Seconda riga:
    • Bottone play: suona il file dall'inizio
    • Bottone pausa: mette in pausa il player o ripremuto riparte dalla pausa
    • Bottone stop: mette in stop il player
    • Bottone delete: elimina dalla playlist i file selezionati nell'are playlist
    • Visualizzazione tempo rimanente del file corrente in minuti:secondi
    • Barra di progresso: visualizza e imposta la posizione corrente nel tempo del file corrente. L'impostazione può essere fatta con il mouse.
  • Tabella playlist: presenta le colonne:
    • ID: identificativo univoco del file; normalmente a fondo verde, rosso evidenzia il file corrente
    • Len: durata in minuti e secondi del file; normalmente verde chiaro, arancio quando il file è "multichannel"
    • Artist: artista del brano letto dai metadati o in sua assenza nome della cartella che contiene il file
    • Title: Titolo del brano letto dai metadati o in sua assenza nome del file
    • Cursore: la posizione del cursore è evidenziata da una riga a fondo blu
Azioni:
  • Cambio brano in esecuzione: doppio click con il mouse sulla riga del nuovo brano oppure posizionare il cursore sul nuovo brano e usare il bottone Play.
  • Cambio posizione di esecuzione del brano corrente: usare il mouse sulla barra di progresso.
Uso della Tastiera: alcuni bottoni sono associati a tasti della tastiera:
  • Bottone Play: tasto "invio",tasto "invio" del tastierino numerico
  • Bottone Pause: tasto ".", tasto "canc" del tastierino numerico
  • Bottone Stop: tasto "delete", tsto "ins" del tastierino numerico
  • Barra di progresso: freccia a sinistra arretra e freccia a destra avanza, di 10 secondi (anche del tastierino numerico)
  • Cursore: freccia in basso successivo (da premere ripetutamente per visualizzare il cursore se non già presente) cursore sul brano successivo, freccia in alto cursore sul brano precedente (anche del tastierino numerico)
